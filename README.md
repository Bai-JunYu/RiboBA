# RiboBA

RiboBA is an R package for bias-aware ORF detection from ribosome profiling (Ribo-seq) data.

## Citation

If you use RiboBA, please cite:

Bai, Junyu, and Ruolin Yang. “RiboBA: A Bias-Aware Probabilistic Framework for Robust ORF Identification across Diverse Ribosome Profiling Protocols.” Preprint, bioRxiv, March 19, 2026. https://doi.org/10.64898/2026.03.17.712439.

## Requirements

### R

* R >= 4.5
* R package dependencies listed in `DESCRIPTION` under `Imports` and `Suggests`

### External tools

Step 2 (`prepare_bam()`) requires Linux/macOS (or Windows via WSL2). Native Windows cmd/PowerShell is not supported.
Windows users can skip Step 2 if they already have `*_cds_txsorted.bam(.bai)` and `*_lncrna_txsorted.bam(.bai)` aligned to Step 1 transcript references (`longest_txs.fa`, `lncRNA_txs.fa`).

* `bowtie` / `bowtie-build` (Bowtie 1.3)
* `samtools` (recommended >= 1.10)
* `curl` (recommended >= 7.0)

## Installation

```r
options(repos = c(CRAN = "https://cloud.r-project.org"))
install.packages("pak")
pak::pkg_install("Bai-JunYu/RiboBA")
```

## Example (Human GRCh38)

### Step 1. Build annotation resources (once per genome/annotation)

```r
library(RiboBA)

build_annotation_index(
  output_dir  = "/path/to/annotation",  # directory for annotation outputs
  genome_file = "/path/to/GRCh38.fa",   # reference genome FASTA
  gff_file    = "/path/to/GRCh38.gtf"   # annotation GTF/GFF
)
```

### Step 2. Prepare transcriptome-aligned BAM files

```r
prepare_bam(
  workdir      = "/path/to/workdir",     # working directory for indices, logs, and BAM files
  fastq_dir    = "/path/to/fastq",       # directory containing adapter-trimmed FASTQ files
  annotate_dir = "/path/to/annotation",  # same output_dir used in build_annotation_index()
  genome_file  = "/path/to/GRCh38.fa",   # same reference genome FASTA used above
  species      = "Homo sapiens",         # species name used by the script
  rrna_fa      = "/path/to/rRNA.fa",     # rRNA FASTA for depletion
  threads      = 16
)
```

### Step 3. Run the RiboBA pipeline

```r
res <- run_riboba_pipeline(
  transcript.file = "/path/to/annotation/transcript/tx_information.RData",  # generated by build_annotation_index()
  bam_dir         = "/path/to/bam"                                          # directory containing prepared BAM files
)
```
